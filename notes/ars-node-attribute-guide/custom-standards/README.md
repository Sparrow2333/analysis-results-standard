# 自定义标准

这个目录预留给未来补充项目自定义标准、团队口径或 ARS 之外的扩展约定。

建议以后按主题拆分，例如：

- `display-layout.md`：表格 shell、行列、缩进、分页、字体、对齐等渲染约定。
- `naming-conventions.md`：`id`、`name`、`label` 的命名规则。
- `analysis-patterns.md`：常见分析模式，例如 AE summary、demographics、labs、shift table。
- `custom-fields.md`：项目自定义字段、扩展字段或 sponsor-defined terminology。

补充自定义标准时，建议每条规则包含：

- 规则名称
- 适用对象或属性
- 使用原因
- 正例
- 反例或常见误用
- 是否映射到官方 ARS 字段
